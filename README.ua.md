# 💆 Застосунок Масажного Салону

Повний набір інструментів для управління масажним салоном, що включає мобільний додаток для клієнтів та майстрів, десктопний додаток для менеджменту та надійний бекенд на .NET із PostgreSQL.

## 🚀 Швидкий старт (Windows)

Найпростіший спосіб почати роботу — скористатися автоматичним скриптом налаштування.

1. **Попередні вимоги**:
   - [Node.js](https://nodejs.org/) (v20+)
   - [Docker Desktop](https://www.docker.com/products/docker-desktop/)
   - [.NET SDK 10+](https://dotnet.microsoft.com/download) (для розробки бекенду)

2. **Запуск налаштування**:
   Двічі клацніть на `setup-windows.bat` у кореневій папці. Це дозволить:
   - **Динамічна IP-адреса**: Визначити вашу локальну IP-адресу та автоматично налаштувати всі файли `.env`.
   - **Frontend**: Встановити залежності та згенерувати API-клієнт Orval.
   - **Desktop**: Встановити залежності та згенерувати клієнт Prisma.
   - **Інфраструктура**: Автоматично запустити Базу Даних та Бекенд через Docker.

## 📁 Структура проєкту

- `/frontend`: Мобільний додаток Expo (React Native) для клієнтів та майстрів.
- `/desktop`: Десктопний додаток Electron + Vite + Prisma для управління.
- `/backend`: Web API на .NET Core для даних та бізнес-логіки.
- `/db-init`: SQL-скрипти для ініціалізації та наповнення бази даних.

## 🛠️ Команди для ручної розробки

Якщо ви віддаєте перевагу ручному запуску компонентів:

### 📱 Мобільний додаток (Frontend)
```bash
cd frontend
npm install
npm start
```

### 💻 Десктопний додаток
```bash
cd desktop
npm install
npx prisma generate
npm run start
```

### ⚙️ Бекенд та База Даних (Інфраструктура)
```bash
docker-compose up -d --build
```

## 🌍 Конфігурація середовища
Усі сервіси покладаються на файли `.env`. Скрипт `setup-windows.bat` створює їх автоматично на основі `.env-template`.

> [!IMPORTANT]
> Параметр `APP_URL` у `.env` повинен бути вашою локальною мережевою IP-адресою (наприклад, `192.168.1.5`), щоб фізичні мобільні пристрої могли підключитися до бекенду. Автоматичний скрипт налаштування зробить це за вас!
